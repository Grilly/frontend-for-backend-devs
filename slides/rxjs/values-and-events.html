RxJS

Values and events

Two dimensions (1/5)

Reactive programming works in two "dimensions":

  • spatial (values in space/memory)
  • temporal (events over time)

Array comparison (2/5)

An observable is similar to a plain JS Array:


      of(1, 2, 3).pipe(
        filter((nr) => nr > 1),
        map((nr) => nr * 10),
        scan((acc, nr) => acc + nr, 0)
      )
      .subscribe((result) => {
        console.log(result);
      });
      // 1
      // 2
      // 3// 2
      // 3// 20
      // 30// 20
      // 50
    

      const result = [1, 2, 3]
        .filter((nr) => nr > 1)
        .map((nr) => nr * 10)
        .reduce((acc, nr) => acc + nr, 0);

      console.log(result);


      // [1, 2, 3]// [2, 3]// [20, 30]// 50
  • Values are handled individually
  • Values may appear over time
  • Functionality is composed
  • Values are handled as a list
  • Values are always static
  • Functionality is chained

Nesting (3/5)

Both observables and arrays can be nested:


      of(of(1), of(2)).pipe(
        flatMap((val) => val)
      )
      .subscribe((result) => {
        console.log(result);
      });

      // 1
      // 2
    

      const result = [[1], [2]]
        .flatMap((val) => val);


      console.log(result);


      // [1, 2]
