Scope (1/4)

  • JS has these kinds of scope:
    • Global scope
    • Module scope ES2015
    • Block scope ES2015
    • Function scope
  • JS has 3 ways to declare variables:
    • var (function scope)
    • const ES2015 (block scope)
    • let ES2015 (block scope)
  • There's no reason to use var anymore.
